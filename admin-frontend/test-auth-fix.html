管理员端认证修复测试

1. 检查当前 localStorage 状态

2. 测试 storage 工具

3. 说明

修复内容：将 useAuth.js 中的 localStorage 直接调用改为使用 storage 工具

问题原因：

  • useAuth.js 使用 localStorage.setItem(key, value) 存储 token（纯字符串）
  • request.js 使用 storage.get(key) 读取，会尝试 JSON.parse
  • JSON.parse 纯字符串 token 失败，返回 null
  • 请求没有 Authorization header，后端返回 401
  • 会话被清除，提示需要登录

解决方案：

  • 统一使用 storage 工具，确保存储和读取方式一致
  • storage.set 会自动 JSON.stringify
  • storage.get 会自动 JSON.parse
